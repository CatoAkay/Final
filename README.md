# Final Project

Idé 1 :
Når lærere legger ut forelesningsnotater og linker til relevante linker. Så hadde det vært fint med "tagger" og/eller kort oppsummering på hva det er man 
skal lese om.
Videre så kan man lage en side som man kan legge inn søkeord på, hvor man får opp resultater basert på sitt søk. 
På den måten slipper eleven frustrasjon momentet i å lese en eller flere tekster/dokumenter. 
Med tanke på de som er fargeblinde kan man legge ved kommentar på hvilken farger som er brukt. 
De som har konsentrasjonvansker kan få beskjed om å ta pause.
Siden hver enkelt bruker har egen profil, så kan man legge opp til relevante sider/linker/dokumenter fra samme lærer (eller andre) 
med informasjon som kan relateres til søket som ble gjort.
Litt som en smart Google for institusjoner. Hvor hver student har en profil.
